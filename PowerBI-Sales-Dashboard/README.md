# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using a Superstore dataset. It provides insights into sales performance, product trends, and regional distribution to support data-driven decision-making.

---

## 📈 Key Features

* Interactive dashboard with 4 pages
* KPI metrics: Total Sales, Total Profit, Total Orders
* Sales trend analysis over time
* Top & Bottom product analysis
* Regional and state-wise performance

---

## 🧠 DAX Functions Used

* SUM → Used to calculate Total Sales and Total Profit
* COUNTROWS → Used to calculate Total Orders
* DIVIDE → Used to calculate Profit Margin
* TOTALYTD → Used to calculate Year-to-Date Sales

---

## 📊 Dashboard Pages

### 1. Overview Dashboard

* Displays key KPIs such as Total Sales, Profit, and Orders
* Sales distribution by region and category

### 2. Sales Analysis

* Sales trends over time using line charts
* Category and sub-category analysis

### 3. Product Performance

* Top 10 and Bottom 10 products by sales
* Profit analysis by product

### 4. Regional Analysis

* Sales distribution by state (map visualization)
* Sales by state and profit by region

---

## 🛠 Tools Used

* Power BI
* DAX
* Excel / CSV Dataset

---

## 🎯 Insights

* Technology category leads in overall sales
* Sales show steady growth over time
* Top products contribute significantly to revenue
* Regional performance varies across different states
