# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using a Superstore dataset. It provides insights into sales performance, product trends, and regional distribution to support data-driven decision-making.

---

## 📈 Key Features

* Interactive dashboard with 4 pages
* KPI metrics: Total Sales, Total Profit, Total Orders
* Sales trend analysis over time
* Top & Bottom product analysis
* Regional and state-wise performance

---

## 🧠 DAX Functions Used

* SUM → Used to calculate Total Sales and Total Profit
* COUNTROWS → Used to calculate Total Orders
* DIVIDE → Used to calculate Profit Margin
* TOTALYTD → Used to calculate Year-to-Date Sales
